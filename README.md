# Day 13 - Expense Search & Analyzer (GUI + Data Logic)

A Python GUI-based application that allows users to manage, search, and analyze expenses.

## 🚀 Features
- Add and delete expenses
- Persistent data storage using JSON
- Search expenses by category
- Filter displayed results dynamically
- Calculate total spending for selected category

## 🧠 Concepts Learned
- Data filtering techniques
- Basic data analysis (aggregation)
- String parsing and data extraction
- GUI + backend integration
- JSON data persistence

## 🛠️ Tech Stack
- Python
- Tkinter (GUI)
- JSON

## ▶️ How to Run
```bash
python app.py
